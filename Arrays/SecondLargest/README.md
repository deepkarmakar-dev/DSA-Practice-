# Array Problems (C++)

This folder contains array-based Data Structures and Algorithms problems implemented in C++.

---

## Problems

### 1. Second Largest Element
- Find the second largest element in an array.
- Returns -1 if second largest element does not exist.

---

## Approach Used
- Traversing array using single pass
- Tracking largest and second largest element

---

## Time Complexity
O(n)

## Space Complexity
O(1)

---

## Language Used
C++
